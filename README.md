# ImageColorizer :
A convolutional neural network based image colorizer architecture.
1. This project showcases how we can transform B&W images to coloured image.
2. Techniques used :
a) AutoEncoders.
b) Transferlearning.
3. Technologies used :
a) skimage.
b) Tensorflow.
c) keras.
d) numpy
